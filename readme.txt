1. MongoDB installieren

2. Mit mongorestore Daten importieren: 
	-CMD-Befehl 'mongorestore /db:questionDatabase /dir:C:/.../database' in mongo Verzeichnis ausführen

3. MongoDB starten (im MongoDB/server/x.x/bin Ordner, x.x entspricht der aktuellen Mongo-version): 
	-mongod 

3. node js Server in einer neuen Konsole mit folgendem Befehl starten:
	- node server.js 

4. Die Webseite ist nun erreichbar unter localhost:8765/site
